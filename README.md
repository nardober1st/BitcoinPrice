# Tecnologias
* Kotlin
* Layouts
    * ConstraintLayout
    * SwipeRefreshLayout
- Componentes UI
    - LineChart
    - TextView
    - Vectors
    - Button
    - EditText
    - ProgressBar
- Snackbar para mostrar mensagem
- SplashScreen
- Consumo de RESTFul API com Retrofit
- ViewBinding
- MVVM clean archictecture
    - ViewModel
    - LiveData
    - Repository pattern
- Offline First
   - ROOM Database (para armanezar dados da API e mostrar para o usuario)
